# Examples Index

This directory contains example task contracts and downstream results.

## Handoff payloads
- `memo-draft-handoff.json` — low-risk document drafting flow
- `email-draft-handoff.json` — high-risk external communication flow
- `research-handoff.json` — research synthesis flow
- `coding-handoff.json` — code review / release-note preparation flow

## Downstream results
- `document-result.json` — successful document draft result
- `action-preview-result.json` — approval-gated action preview result
- `blocked-action-result.json` — blocked high-risk action result
- `partial-document-result.json` — partial document result with missing inputs

## Suggested reading order
1. `memo-draft-handoff.json`
2. `document-result.json`
3. `email-draft-handoff.json`
4. `action-preview-result.json`
5. `blocked-action-result.json`
6. `research-handoff.json`
7. `coding-handoff.json`
